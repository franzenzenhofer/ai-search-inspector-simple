## üîÑ REGENERATING ASSETS

If you need to regenerate any assets:

### Regenerate All Images
```bash
source venv/bin/activate
python generate_images.py
```

### Regenerate Icons Only
```bash
cd store-assets/icons
rsvg-convert -w 16 -h 16 icon.svg -o icon16.png
rsvg-convert -w 32 -h 32 icon.svg -o icon32.png
rsvg-convert -w 48 -h 48 icon.svg -o icon48.png
rsvg-convert -w 128 -h 128 icon.svg -o icon128.png
cp icon*.png ../..
```

### Edit SVG Icon
1. Open `icons/icon.svg` in any SVG editor
2. Make changes
3. Run the commands above to regenerate PNGs
